При выполнении задания можно воспользоваться примером - класс Cleaner
1. Прочитать о конструкторах - https://javarush.ru/groups/posts/konstruktory-v-java-1
2. В каталоге src/users создать класс Tester, унаследовать его от класса User.
3. В классе Tester переопределить метод родительского класса sayProfession(). Реализация метода на ваше усмотрение.
4. В классе Tester имплементировать интерфейс Work и реализовать методы этого интерфейса.
5. В каталоге src/users создать класс Manager, унаследовать его от класса User.
6. В классе Manager переопределить метод родительского класса sayProfession(). Реализация метода на ваше усмотрение.
7. В классе Manager имплементировать интерфейс Work и реализовать методы этого интерфейса.
8. В классе Manager имплементировать интерфейс Manage и реализовать методы этого интерфейса.
9. В методе main класса Main создать экземпляры Tester и Manager и повызывать у них методы, аналогично экземпляру Cleaner.
У тестеров по 2 метода из интрфейса Work, у менеджеров 2 метода из интрфейса Work и 1 из интрфейса Manage.

Бонусное задание: создать еще какой нибудь интерфейс с парой методов и еще какого нибудь работника, который наследуется от User
и имплементирует этот ваш интерфейс.
